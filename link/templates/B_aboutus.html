ကျွန်တော်တို့အကြောင်း

စွန့်ပစ်ပစ္စည်းများကို လျှော့ချရန်၊ ပြန်လည်အသုံးပြုရန်နှင့် ကျန်းမာသောပတ်ဝန်းကျင်ကို အကျိုးပြုရန်အတွက် လူတစ်ဦးချင်းစီနှင့် စီးပွားရေးလုပ်ငန်းများကို အခွင့်အာဏာပေးသည့် မြန်မာနိုင်ငံတွင် ထိပ်တန်းပလက်ဖောင်းဖြစ်လာစေရန်ရည်ရွယ်သည်။

ဘာလုပ်ရမလဲ

Recyclelink သည် ပစ္စည်းများကို ပြန်လည်အသုံးပြုခြင်းနှင့် ပြန်လည်အသုံးပြုခြင်းအတွက် ပြီးပြည့်စုံသော ဖြေရှင်းချက်တစ်ခုဖြစ်သည်။ ကျွန်ုပ်တို့သည် အဓိကဝန်ဆောင်မှုသုံးခုကို ပေးဆောင်ပါသည်။

ပြန်လည်အသုံးပြုနိုင်သော ပစ္စည်းများကို ဝယ်ယူခြင်း။

ကျွန်ုပ်တို့သည်ပလက်စတစ်အိတ်များ၊ပုလင်းများ၊စာရွက်များ စသည့်ပြန်လည်အသုံးပြုပစ္စည်းများကိုဝယ်သည်

ဘာကြောင့်ကျွန်တော်တို့ကိုရွေးချယ်သင့်လည်း

ရေရှည်တည်တံ့စေရန်

ကျွန်ုပ်တို့သည် အမှိုက်များကို လျှော့ချရန်နှင့် ecofamily အလေ့အကျင့်များကို မြှင့်တင်ရန် ကတိပြုပါသည်။

အဆင်ပြေသောဖြေရှင်းနည်းများ

ပေးပို့ခြင်းဝန်ဆောင်မှုများနှင့် လွယ်ကူသောဆက်သွယ်ရေးဖြင့် ကျွန်ုပ်တို့သည် ပြန်လည်အသုံးပြုခြင်းကို လွယ်ကူစွာပြုလုပ်ပါသည်။

လူမှုအသိုင်းအဝိုင်းများကို အားကောင်းစေခြင်း။

ပြန်လည်အသုံးပြုခြင်း သည် စုပေါင်းလုပ်ဆောင်မှုကို အားပေးသည့် သဘာဝပတ်ဝန်းကျင်ကို ဂရုစိုက်သောသူများကို ချိတ်ဆက်ပေးသည်။

အဖွဲ့၀င်များ

ခင်မြတ်သူ

ခင်မြတ်သူသည် စီမံခန့်ခွဲမှု မန်နေဂျာဖြစ်သည်။ လုပ်ငန်းများကို စနစ်တကျ ဆောင်ရွက်နိုင်ရန် ကြပ်မတ်ပြီး အသင်းအဖွဲ့ကို ဦးစီးပြုလုပ်သည်။ ပရောဂျက်စီမံခန့်ခွဲမှုနှင့် သဘာဝပတ်ဝန်းကျင် ထိန်းသိမ်းမှု အတွေ့အကြုံများဖြင့် ပြန်လည်အသုံးပြုမှု (Recycling) ကို လူတိုင်း အသုံးချနိုင်ရန် ကြိုးပမ်းဆောင်ရွက်နေသည်။

လင်းလဲ့ဖြူ

လင်းလဲ့ဖြူသည် အဓိက Developer ဖြစ်ပြီး ဝဘ်ဆိုဒ် ဖန်တီးမှုနှင့် ထိန်းသိမ်းမှု တာဝန်ယူထားသည်။ ဝယ်သူများနှင့် ရောင်းသူများအတွက် အသုံးပြုရလွယ်ကူစေရန် ဝဘ်ဆိုဒ်ပေါ်တွင် User-Friendly Design ကို ထိန်းသိမ်းဆောင်ရွက်သည်။ Web Development နှင့် Database Management မှာ ကျွမ်းကျင်သောကြောင့် Platform ကို သာမန်ပြုလုပ်၍ ထိန်းသိမ်းနိုင်သည်။

ဆုမြတ်သန္တာ

ဆုမြတ်သန္တာသည် ဈေးကွက်ဖန်တီးမှုနှင့် ဖောက်သည်ဝန်ဆောင်မှု တာဝန်ယူထားသည်။ Platform ကို ကြော်ငြာပြီး အသုံးပြုသူများနှင့် ချိတ်ဆက်ကာ ပြသာနာများ ဖြေရှင်းပေးသည်။ Social Media Campaign များကို စီမံပြီး ပြန်လည်အသုံးပြုမှု (Recycling) ၏ အရေးကြီးမှုကို လူထုထံသို့ ရှင်းလင်းပေးနေသည်။

Recycle Link ဆိုတာဘာလဲ?

Recycle Link က ပြန်လည်အသုံးပြုနိုင်သော ပစ္စည်းများ (ပလတ်စတစ်ပုလင်း၊ စက္ကူစသည်) ကို ရောင်းဝယ်နိုင်မယ့် အွန်လိုင်းပလက်ဖောင်းတစ်ခု ဖြစ်ပါတယ်။ Admin များ က ရောင်းသူများထံမှ ပြန်အသုံးပြုနိုင်သော ပစ္စည်းများ ကို ဝယ်ယူပြီး၊ ဝယ်သူများအတွက် ပြန်လည်ထုတ်လုပ်ထားသော eco-friendly ပစ္စည်းများ ကိုလည်း ရောင်းချပေးပါသည်။

ပြန်အသုံးပြုနိုင်သော ပစ္စည်းတွေကို ဘယ်လို ရောင်းမလဲ?

အသုံးမလိုတော့တဲ့ recyclable ပစ္စည်းများ ကို ဓာတ်ပုံတင်ပြီး၊ စျေးနှုန်း သတ်မှတ်ပြီး Admin ကို တင်ပြနိုင်ပါတယ်။ Admin မှ အတည်ပြုပြီးနောက် သင့်ပစ္စည်းများကို တန်ဖိုးအလိုက် ဝယ်ယူမည် ဖြစ်သည်။

ပြန်လည်အသုံးပြုထားတဲ့ ဘယ်လိုပစ္စည်းများကို ဝယ်နိုင်မလဲ?

Recycle Link မှာ ပြန်လည်အသုံးပြုထားသော eco-friendly ပစ္စည်းများ ဝယ်ယူနိုင်သည်။ Reusable Bags, ပလတ်စတစ်မှ ပြုလုပ်ထားသော ဆင်တူပစ္စည်းများ, furniture နှင့် အလှဆင်ပစ္စည်းများ ပါဝင်ပါသည်။

ငွေပေးငွေယူလုပ်ငန်းစဉ် ဘယ်လိုဖြစ်မလဲ?

ပစ္စည်းရောင်းချသူများ အတွက် - ဝယ်ယူပြီးသည့်နောက် ငွေသားသို့မဟုတ် အွန်လိုင်းငွေပေးချေမှု ဖြင့် ငွေပေးချေခြင်း ပြုလုပ်မည်။ ဝယ်သူများ အတွက် - အွန်လိုင်းငွေပေးချေမှု သို့မဟုတ် ငွေချေမှသာပစ္စည်းပို့ခြင်း (COD) ဖြင့် ဝယ်ယူနိုင်သည်။

ပြန်လည်အသုံးပြုနိုင်သောပစ္စည်းများအတွက်အရည်အသွေး စစ်ဆေးမှုရှိလား?

ဟုတ်ပါတယ်။ Admin များ မှ recyclable ပစ္စည်းများ၏ အရည်အသွေး နှင့် ပြန်အသုံးပြုနိုင်မှု ကို စစ်ဆေးပြီးမှ ဝယ်ယူမည် ဖြစ်သည်။

Recycle Link ကို ဘာကြောင့် အသုံးပြုသင့်တာလဲ?

ပတ်ဝန်းကျင်ထိန်းသိမ်းမှု အတွက် အထောက်အကူဖြစ်စေသည်။ အမှိုက်တွေကို လျှော့ချနိုင်ပြီး, အသုံးမလိုတဲ့ ပစ္စည်းတွေကို ပြန်အသုံးပြုနိုင်အောင် ကူညီပေးသည်။ ငွေရှာနိုင်ခြင်း နှင့် eco-friendly ပစ္စည်းများ ဝယ်ယူနိုင်ခြင်း တို့ကို လွယ်ကူစွာ ပြုလုပ်နိုင်သည်။
